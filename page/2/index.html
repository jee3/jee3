按位操作符

按位操作符的 MDN 讲解

按位操作符（Bitwise operators）将其操作数（operands）当作 32 位的比特序列（由 0 和 1 组成），而不是十进制、十六进制或八进制数值。例如，十进制数 9，用二进制表示则为 1001。按位操作符操作数字的二进制形式，但是返回值依然是标准的 JavaScript 数值。

tips：异或 ⊕ 和同或 ⨀ 是反函数（异或是俩不一样为 1，同或就是俩一样为 1）
